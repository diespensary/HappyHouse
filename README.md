# HappyHouse
HappyHouse - веб-приложение для мебельного магазина. 

Приложение состоит из следующих страниц:
- Страница авторизации/регистрации
- Главная страница, на которой отображены товары
- Страница корзины
- Страница профиля
- Страница истории заказов
### Приложение обладает следующим функционалом:
- Авторизация/регистрация
- Просмотр и изменение профиля
- Просмотр и редактирование корзины
- Оформление заказа
- Просмотр истории заказов
